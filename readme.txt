Conditionally hide BuddyPress XProfile Field Groups from the user based on ther role.

== Description ==
This is a simple plugin used to hide XProfile Field groups from displaying to different user types based on the user's role. To hide a field group, go to Users -> Groups Display in the wp-admin and check appropriate box in the grid.


== Installation ==

= Manual Installation =

1. Upload the entire `/buddypress-role-field-groups` directory to the `/wp-content/plugins/` directory.
2. Activate BuddyPress Role Field Groups through the 'Plugins' menu in WordPress.
